## swift enum

- enumは、namespaceとしても使える
- 定数定義のみしたい場合、StructのConstよりも、enumで定義する方が、initializerする必要ない点で好ましい

＜例＞

```
struct Const {

static let test = ""
static let test = ""
static let test = ""
static let test = ""

}

↓

enum Const {

static let test = ""
static let test = ""
static let test = ""
static let test = ""

}

```
