## [swift] failable initialzer

初期化を保証できない場合は、下記のようにinit?としてあげて、初期化処理内部で初期化できない際に `return nil` をしてあげると良い

```

class Animal {

 let name: String

 init?(name: String) {
  if name.count > 3 {
    return nil
  }
  self.name = name
 }

}

```
