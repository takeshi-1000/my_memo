## [c] プロトタイプ宣言

- 　下記のようなコードの場合エラーになる。
- 　main関数よりも上でsayHiメソッドを定義する必要がある

```

int main(void) {
 sayHi();
 return 0;
}

void sayHi() {
  printf("hi!");
}

```

- メソッドの位置をずらさずに、コンパイルエラーにならないようにするには、プロトタイプ宣言を行う
- 処理の内容は書かず、メソッドだけ先に宣言しとくみたいな感じ

```

// プロトタイプ宣言
void sayHi();

int main(void) {
 sayHi();
 return 0;
}

void sayHi() {
  printf("hi!");
}

```
