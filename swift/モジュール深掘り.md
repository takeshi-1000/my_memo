### 実験3

hoge.swift側をモジュール化してみる

※ここでソースコードを下記のように変更

```
main.swift

import Hoge

print("Hello")
let _ = Hoge()

```

```
hoge.swift

public class Hoge {

  public init() {
     print("Hoge init")
  }
}

```
↑ public にする理由深掘り
